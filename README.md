# GitHub 個人網站

這是我的 GitHub Pages 個人網站。

## 使用方式

1. 將此資料夾重新命名為 `<你的GitHub用戶名>.github.io`
2. 在 GitHub 上建立同名的 repository
3. 將檔案推送到該 repository
4. 網站將自動部署在 `https://<你的GitHub用戶名>.github.io`

## 目錄結構

```
github-pages/
├── index.html      # 主頁
├── css/
│   └── style.css   # 樣式表
├── js/
│   └── main.js     # JavaScript
├── images/         # 圖片資料夾
├── assets/         # 其他資源
└── README.md       # 說明文件
```
